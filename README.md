# 停车场管理系统 java1115

## 项目概述

停车场管理系统是一套功能齐全的管理解决方案，支持停车场运作的全面管理，包括停车场、会员及违规处理等多个功能模块。

## 技术栈

本项目采用以下技术构建：

- **前端**：Vue3
- **后端**：Django

## 功能模块

### 停车场管理
- 停车场信息管理
- 车位状态实时更新

### 会员管理
- 会员资料维护
- 会员积分管理

### 违规管理
- 违规处理流程
- 违规记录查询

## 系统角色

系统包括以下角色：
- 管理员：全面管理系统的各项功能
- 员工：执行日常操作，如车辆入场、出场管理等
- 会员：查看个人信息，进行积分兑换等

## 运行环境

- **后端数据库**：支持 SQLite、MySQL
- **服务器环境**：与 Django 和 Vue3 兼容的常规服务器环境

## 目录结构

```
 project/
 ├── frontend/             # 前端Vue3项目目录
 └── backend/              # 后端Django项目目录
     ├── manage.py         # Django项目管理脚本
     ├── app/              # Django应用目录
     └── ...
```

## 部署步骤

1. 准备后端数据库环境（SQLite 或 MySQL）
2. 部署后端 Django 应用
3. 部署前端 Vue3 应用
4. 进行系统配置和测试

## 核心亮点

- **前后端分离架构**：采用 Django + Vue3 架构，易于维护和扩展
- **功能齐全**：覆盖停车场管理的各个方面
- **系统稳定高效**：已在本地环境中成功运行，确保系统稳定性
- **可视化图表**：直观展示数据，方便用户分析
